wait和sleep 深度分析

2016-12-05
  • java

首先我们需要明白wait与sleep的区别，区别大致有下面几个方面:

  1. wait是Object中的方法,sleep是Thread中的方法
  2. wait必须在同步代码块中调用，也就是说必须首先获得监视器锁，然后才能调用,sleep可以在任何可能的地方调用
  3. wait释放锁,sleep不释放锁
  4. wait尽可能的少消耗cpu资源,sleep消耗更多的cpu资源
